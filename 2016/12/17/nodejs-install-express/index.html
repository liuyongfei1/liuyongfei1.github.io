nodejs新手小笔记之-express安装

express概述

Express 是一个基于 Node.js 平台的极简、灵活的 web 应用开发框架，它提供一系列强大的特性，帮助你创建各种 Web 和移动设备应用。使用express我们可以快速的搭建一个网站。
很多流行的nodejs框架都是基于express，例如：Feathers，ItemsAPI
KeystoneJS等。
